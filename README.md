
# React Learning Journey

This repository documents my **React.js learning journey** through mini projects. It showcases my progress from basic React concepts to building functional apps with state management, hooks, and local storage. Each project focuses on different aspects of React development.

---

## Projects

### 1. **Daily Study Planner**
- A simple task management app for students.
- Features:
  - Add, complete, and delete tasks.
  - Persistent storage using `localStorage`.
  - Minimalistic and clean UI.
- [Live Demo](https://react-learning-journey-eight.vercel.app/DailyStudyPlanner)

### 2. **Counter App**
- Basic counter with increment, decrement, and reset functionality.
- Helps understand state management and event handling in React.

### 3. **Expense Tracker**
- Tracks daily expenses.
- Features:
  - Add and remove transactions.
  - Calculate balance, income, and expense.
  - Local storage persistence.

### 4. **Job Application Tracker**
- Helps manage and track job applications.
- Features:
  - Add new applications.
  - Update status.
  - Remove applications.

### 5. **Simple Todo App**
- Basic to-do list app.
- Features:
  - Add, mark as complete, and delete tasks.
  - Demonstrates state handling and list rendering.

### 6. **Toggle Theme**
- Simple app to switch between dark and light modes.
- Focuses on theme management and CSS variables in React.

### 7. **Weather App**
- Displays weather information for a city.
- Features:
  - Fetch data from an API.
  - Display temperature, humidity, and weather description.
  - Dynamic styling based on weather conditions.

---

## Tech Stack

- **Frontend:** React.js
- **State Management:** React `useState`, `useEffect`
- **Styling:** CSS
- **Data Storage:** `localStorage` for mini-projects
- **API Integration:** For Weather App

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SmrutiPatil12/react-learning-journey.git
````

2. Navigate to the project folder, then to the specific project folder:

   ```bash
   cd react-learning-journey/DailyStudyPlanner
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the development server:

   ```bash
   npm start
   ```
5. Open your browser at `http://localhost:3000`.
